【指令第一段：設定角色與目標】

「請你擔任一位專業的兒童互動教材設計師與前端工程師。我要製作一款給 [國小一年級] 學生使用的 [英文科] 互動學習 Web App。 請使用html+css+js單一檔案撰寫，並搭配 Tailwind CSS 進行美化。」

【指令第二段：視覺風格】

「視覺風格請採用 『清新美肌風格』：

色調要明亮柔和（如天空藍、嫩綠、奶油黃）。

使用大量的圓角卡片設計，介面要乾淨無雜質。

使用圖示來代替複雜的圖片。

整體要有童趣，像是一個冒險遊戲的大廳。」

【指令第三段：功能需求 (系統架構)】

「請製做國小一年級英文學習的互動遊戲，內容需包含以下功能：

遊戲大廳：要有可愛的 NPC (如雲朵寶寶) 引導，並有傳送門通往不同關卡。

互動關卡：請設計 3-4 個小遊戲，範圍是 words.csv 中的單字，例如：
- 聽音辨識（聽到單字後選擇正確的圖片）
- 連連看（配對單字與圖片）
- 記憶力遊戲（快速閃現單字）
- 拼字遊戲（拖曳字母組成單字）
- 選擇題（單字與圖片配對）

測驗系統：設計是非題或選擇題，答錯要有引導，答對要有回饋。

獎勵系統：設計一個『收集冊』，過關可以獲得貼紙。

音效系統：請使用 Web Audio API 製作簡單的音效（點擊聲、成功聲、失敗聲），不需要外部音檔。」
